## üöÄ Installation & Usage

### Option 1: Run Locally (Easiest)
Since UnitMaster Ultimate is a **Single File Application**, no installation or build process (npm/yarn) is required!

1. Download the `index.html` file.
2. Double-click it to open in your default web browser.
3. Enjoy! (Internet connection required initially for Tailwind/FontAwesome CDNs).

### Option 2: Host it
Simply upload the `index.html` file to GitHub Pages, Netlify, or Vercel.
